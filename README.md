# 🌍 Wanderlust – Explore & Book Amazing Stays  

**Wanderlust** is a full-stack web application for discovering and booking unique stays around the world. Built with **Node.js, Express.js, MongoDB, HTML, CSS, and JavaScript**, it provides a seamless and responsive user experience.  

🔗 **Live Demo:** [Wanderlust Website](https://wanderlust-project-full-stack.onrender.com/listings)  

## 🚀 Features  
- **Browse & Filter Listings** – Easily search and filter stays based on categories.  
- **Responsive Design** – Fully optimized for desktop and mobile devices.  
- **User Authentication** – Secure login & registration system.  
- **Booking & Listing Management** – Users can book stays and manage their own listings.  
- **MongoDB Integration** – Stores user and listing data efficiently.  

## 🛠️ Tech Stack  
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Hosting:** Render
